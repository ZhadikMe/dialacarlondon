FAQ

Στην ενότητα αυτή στοχεύουμε στην κάλυψη γενικών ερωτήσεων που έχουν οι πελάτες.

Μπορώ να λάβω απόδειξη για εργασία με μετρητά;
Ναι, παρακαλώ ρωτήστε απευθείας τον οδηγό αν θέλετε απόδειξη για εργασία με μετρητά. Η απόδειξη μετρητών μεταξύ εσάς και του οδηγού δεν είναι απόδειξη ΦΠΑ ή, αν επιθυμείτε, μπορούμε να σας στείλουμε email επίσης.

Χρεώνετε για τον χώρο στάθμευσης αυτοκινήτων στα αεροδρόμια;
Παρακολουθούμε τις ώρες προσγείωσης πτήσεων σε όλα τα μεγάλα αεροδρόμια για να διασφαλίσουμε ότι ο χρόνος στάθμευσης αυτοκινήτου διατηρείται στο ελάχιστο. Για τους πελάτες λογαριασμού, χρεώνουμε τις χρεώσεις στάθμευσης αυτοκινήτου κατά την τιμολόγηση. Για τους πελάτες που πληρώνουν με μετρητά ή πιστωτική κάρτα, προστίθενται αυτόματα £8,00 στην τιμή του ταξιδιού σας, τα οποία καλύπτουν τη χρέωση στάθμευσης του αυτοκινήτου σας συν τα πρώτα 40 λεπτά χρόνου αναμονής. Μετά τα πρώτα 40 λεπτά θα χρεωθείτε για τυχόν πρόσθετες χρεώσεις στάθμευσης ανάλογα.

Ενδιαφέρομαι για το άνοιγμα ενός λογαριασμού, μπορείτε να μου στείλετε τον τιμοκατάλογο των λογαριασμών σας;
Εάν ενδιαφέρεστε να ανοίξετε έναν λογαριασμό σε εμάς, μη διστάσετε να περιηγηθείτε στην ενότητα λογαριασμών μας και να ανακαλύψετε όλα τα οφέλη που συνεπάγονται για τους κατόχους λογαριασμού. Θα χαρούμε επίσης να μιλήσουμε μαζί σας και να σας βοηθήσουμε στη δημιουργία ενός λογαριασμού. Μπορείτε να μας καλέσετε απευθείας στο 020 8563 8888.

Η ώρα της πτήσης μου έχει αλλάξει. Ελέγχετε τις ώρες πτήσεων;
Ναι, το κάνουμε, επομένως όταν κάνετε κράτηση για μεταφορά από το αεροδρόμιο, ζητάμε τα στοιχεία της πτήσης σας. Έχουμε μια εξειδικευμένη ομάδα που ελέγχει τις ώρες των πτήσεων 24 ώρες το 24ωρο και ενημερώνεται για τυχόν γνωστές αλλαγές ώρας. Δεν χρειάζεται να ανησυχείτε για να μας ενημερώσετε, θα στείλουμε αυτόματα το αυτοκίνητό σας για να φτάσει εγκαίρως για την προσγείωση της πτήσης σας.

Πόση χωρητικότητα αποσκευών υπάρχει σε ένα μίνι ταξί;
Για να ελέγξετε τη χωρητικότητα των αποσκευών, κάντε κλικ εδώ.

Μπορώ να ακυρώσω μια κράτηση αφού την έχω κάνει;
Εφόσον ακυρώσετε το όχημά σας εντός εύλογου χρονικού πλαισίου και δεν έχουμε ορίσει οδηγό, δεν θα υπάρξει χρέωση για την ακύρωση. Σε περιπτώσεις όπου ο οδηγός έχει ήδη οριστεί ή έχει διανύσει κάποια απόσταση ή ακόμη και έχει φτάσει στη διεύθυνση παραλαβής, ενδέχεται να χρεωθείτε ακυρωτικό τέλος. Αυτό θα υπολογιστεί με βάση τα χιλιόμετρα που έχουν διανυθεί.

Νομίζω ότι μπορεί να έχω έναν λογαριασμό, αλλά έχω ξεχάσει τις λεπτομέρειες.

Εάν πιστεύετε ότι έχετε λογαριασμό στην Dial a Car αλλά έχετε ξεχάσει τα στοιχεία, καλέστε μας στο 020 8563 8888 ή στείλτε email στο: accounts@dialacarlondon.com και θα σας βοηθήσουμε να βρείτε αυτό που ψάχνετε.

Επηρεάζει η ποσότητα των αποσκευών που μεταφέρω την τιμή του ταξιδιού;

Όχι, δεν υπάρχει επιπλέον χρέωση ανεξάρτητα από τον όγκο των αποσκευών που μεταφέρετε σε ένα από τα οχήματά μας. Ωστόσο, η χωρητικότητα αποσκευών ενδέχεται να είναι περιορισμένη ανάλογα με τον αριθμό των επιβατών που μεταφέρονται στο όχημα.
